Running instructions can be found inside the source code.
Source code is found in the PilotDataCompilter/src path
Data is stored in the Data folder.

1. Import the most recent version of ILS_Pilot_Data_with_all_DGMs before running. This will be a .sav file so it 2. needs to be exported to .csv before running. Put this in the Data folder.
3. Additionally, download the most recent version of "ILS Approach DGMs (Single AOI)". You can replace the existing file by that name in the Data folder.

YOU WILL NEED TO UPDATE THE PATHS IN THE SOURCE CODE. They are full paths and so will not work automatically when you pull this code from github.
